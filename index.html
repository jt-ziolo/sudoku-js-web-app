    Fill in blank squares with the numbers 1 through 9 until all squares are filled and the following rules are met:

    1. Each row only contains one of each number.
    2. Each column only contains one of each number.
    3. Each three-by-three square only contains one of each number.

    Select a square using the mouse or arrow keys. Enter or delete numbers in squares using your keyboard. Hold Ctrl while entering a number to add it as a pencilmark. Have fun!
